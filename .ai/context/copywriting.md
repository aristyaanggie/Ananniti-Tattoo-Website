# Context - Copywriting

Copywriting style guide untuk Ananniti Tattoo Bali.

## Voice & Tone

### Voice (Konsisten)
- Professional namun approachable
- Knowledgeable dan trustworthy
- Passionate tentang tattoo art
- Respectful terhadap customer autonomy
- Clear dan direct

### Tone (Adaptable)
- **Friendly**: Customer interactions, welcomes
- **Professional**: Information pages, policies
- **Inspirational**: Portfolio, testimonials
- **Supportive**: Error messages, help text
- **Urgent**: Promotions, limited-time offers

## Prinsip Penulisan

### 1. Clarity
- Gunakan simple language
- Hindari jargon kecuali necessary
- Short sentences dan paragraphs
- Active voice preferred
- Direct dan specific

### 2. Brevity
- Remove unnecessary words
- Get ke point quickly
- Scannable content
- Gunakan lists dan bullets
- Hindari redundancy

### 3. Authenticity
- Be genuine dan honest
- Show personality
- Tell real stories
- Hindari corporate speak
- Gunakan natural language

### 4. Actionable
- Clear call-to-action
- Tell users what to do
- Gunakan imperative mood
- Specific benefits
- Remove barriers

## Tipe Konten

### Headlines

**Purpose**: Grab attention, convey main idea

**Guidelines**:
- 6-12 words ideal
- Gunakan power words
- Be specific
- Create curiosity
- Include benefit

**Contoh**:
- "Premium Custom Tattoo Design in Bali"
- "Book Your Appointment Today"
- "Discover Your Tattoo Story"

### Subheadings

**Purpose**: Support headline, elaborate idea

**Guidelines**:
- Expand pada headline
- 8-15 words
- Add specific details
- Benefit-focused

**Contoh**:
- "Professional artists dengan 10+ years experience"
- "Safe, sterile, dan stunning results"
- "From concept to skin"

### Body Copy

**Purpose**: Provide information dan persuade

**Guidelines**:
- Focus pada benefits, bukan features
- Gunakan conversational language
- Break into short paragraphs
- Gunakan active voice
- Show, jangan tell

**Contoh**:
```
Instead of: "We use premium ink pigments"
Write: "Your tattoo will stay vibrant for years to come"

Instead of: "3-hour consultation process"
Write: "We spend time understanding your vision"
```

### Buttons & CTAs

**Purpose**: Guide user action

**Guidelines**:
- Action-oriented verbs
- First person when possible
- Specific outcome
- Urgency jika appropriate
- 2-5 words

**Contoh**:
- "Book My Appointment"
- "View Portfolio"
- "Shop Merchandise"
- "Get Started Today"
- "Learn More"

### Error Messages

**Purpose**: Explain what went wrong dan how to fix

**Guidelines**:
- Apologetic tone
- Specific problem
- Clear solution
- Positive language
- Hindari blame

**Contoh**:
```
Bad: "Invalid email address"
Good: "Please enter a valid email address"

Bad: "Form submission failed"
Good: "We couldn't save your information. Please check all fields dan try again"
```

### Success Messages

**Purpose**: Confirm action completed

**Guidelines**:
- Positive, celebratory tone
- Specific confirmation
- Next steps
- Optional recommendation

**Contoh**:
```
"Booking confirmed! Check your email for details"
"Merchandise added to cart"
"Welcome to our newsletter!"
```

### Help Text

**Purpose**: Guide user through process

**Guidelines**:
- Supportive tone
- Specific guidance
- Short dan scannable
- Link to more info
- Encouragement

**Contoh**:
```
"Upload a clear photo, at least 1MB"
"We'll respond within 24 hours"
"Questions? Check our FAQ or contact us"
```

### Form Labels

**Purpose**: Identify input fields

**Guidelines**:
- Clear dan specific
- Hindari abbreviations
- Gunakan standard terms
- Optional indication
- Concise

**Contoh**:
```
Good: "Full Name"
Good: "Booking Date"
Good: "Design Description (optional)"

Hindari: "Name (req'd)"
Hindari: "Date/Time"
```

## Product Descriptions

### Portfolio Items

**Format**: 
- Title: Style + inspiration
- Description: Story dan details
- Medium: Technique used
- Session count: Timeline

**Contoh**:
```
"Japanese Dragon Sleeve"
This traditional Japanese dragon was inspired by the client's heritage. 
Created using classic black ink dengan fine line work throughout.
Medium: Black ink, fine line
Sessions: 5 (20 hours total)
```

### Merchandise

**Format**:
- Title: Clear product name
- Description: What it is, who it's for
- Features: Key benefits
- Specifications: Material, size, etc.

**Contoh**:
```
"Ananniti Studio T-Shirt"
Premium cotton tee featuring our studio logo.
Perfect for tattoo enthusiasts dan art lovers.

Features:
- 100% organic cotton
- Screen printed design
- Unisex fit
- Available in S-XL

Materials: Organic cotton
Care: Machine wash cold
```

## Marketing Copy

### Tagline/Slogan

- Memorable
- Benefit-focused
- Specific to brand
- 3-8 words

**Contoh**:
- "Where Art Meets Skin"
- "Your Story, Our Art"
- "Premium Tattoos in Paradise"

### Promotional Text

- Create urgency
- Highlight benefit
- Clear CTA
- Limited time indicator

**Contoh**:
```
"Book now dan receive a free design consultation
Valid until July 31st"
```

## Technical Guidelines

### Grammar & Style

- Gunakan Oxford commas
- Consistent punctuation
- Hindari exclamation overuse
- Gunakan contractions naturally
- Parallel structure

### Numbers

- Spell out 0-9
- Gunakan numerals 10+
- Gunakan $ untuk currency
- Gunakan % untuk percentages

### Links

- Descriptive anchor text
- Hindari "click here"
- Indicate external links
- Consistent styling

**Contoh**:
```
Good: "Learn more about our booking process"
Bad: "Click here for more info"
```

### Capitalization

- Title Case untuk headings
- sentence case untuk body copy
- UPPERCASE untuk emphasis (sparingly)

## Tone Examples

### Friendly (Customer Welcome)
```
"Welcome to Ananniti! We're so excited to help bring your tattoo vision to life."
```

### Professional (Service Info)
```
"Our sterilization process exceeds industry standards for your safety."
```

### Inspirational (Portfolio)
```
"Every tattoo tells a story. This gallery showcases the stories our clients trusted us to tell."
```

### Supportive (Error Message)
```
"Looks like we had trouble processing your booking. Please try again or contact us for help."
```

## Accessibility

- Hindari color alone untuk convey meaning
- Gunakan descriptive link text
- Provide alt text untuk images
- Gunakan proper heading hierarchy
- Sufficient color contrast
